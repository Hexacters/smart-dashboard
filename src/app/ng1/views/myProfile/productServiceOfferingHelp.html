Help on UNSPSC (United Nations Standard Products and Services Code) Search


Example

A search for "uniform rental" will return all commodities using the terms "uniform" OR "rental".


Returns

91111501 - Uniform rental
53102709 - Ambulance officers uniforms
60105407 - Apartment rental instructional materials


  • Clicking the check box labeled "Search for ALL terms only" and performing a search will find commodities that contain ALL terms entered.

Example

A search for "uniform rental" will return all commodities using the terms "uniform" AND "rental".

Returns

91111501 - Uniform rental


  • Searching for a UNSPSC Commodity Code (number) will return every code that begins with the numbers you enter.

Example

If you enter 321, all codes that begin with the combination 321 will appear in the search results.

Returns

32100000 - Printed circuits and integrated circuits and micro assemblies
32101500 - Circuit assemblies and radio frequency RF components
32101502 - Printed circuit assemblies PCAs
